capnweb-go playground

A Go Cap'n Web server, called from the browser with the reference capnweb client over WebSocket. connecting…

greet — a simple call

add — typed arguments

counter — pass-by-reference stub

NewCounter() returns a live server-side object. Each increment is a call on that remote stub; its state lives on the server.

pipelining — one round trip

api.NewCounter().Add(10) — the counter promise is piped straight into .Add without awaiting it, so both calls go in a single round trip.
